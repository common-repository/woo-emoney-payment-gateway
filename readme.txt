== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the &#039;Plugins&#039; screen in WordPress
3. Use the Settings link in the plugins list page to configure the plugin
